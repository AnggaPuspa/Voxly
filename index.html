Tentang Proyek

Apa itu Voxly?

Proyek ini berawal dari tugas mata kuliah Kecerdasan Buatan, yaitu membuat model sederhana menggunakan Teachable Machine. Dari situ, saya mempunyai ide untuk membuat sebuah model yang dapat mengenali perintah suara 'benar' dan 'salah', yang kemudian saya implementasikan langsung ke dalam sebuah website.

Proyek ini menggunakan Teachable Machine untuk melatih modelnya, TensorFlow.js agar model tersebut bisa berjalan di browser, dan HTML/CSS/JavaScript untuk membangun semua tampilannya.
